# Easy to Use Nest.JS 10 Starter Pack
## Nest.JS 10 Clean Architecture + MySQL + Docker + Swagger Example

## Usage
1. Add .development.env
``` sh
DB_HOST=
DB_PORT=
DB_USER=
DB_PASS=
DB_DATABASE=
```
2. Run App
``` sh
npm install
npm run start
```
3. Build
``` sh
npm run build
```

## Dockerfile
- Service APP with PM2
